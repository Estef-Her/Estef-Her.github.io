Sistema de Tienda

Esta aplicación es una herramienta sencilla que se conecta con themoviedb.org para obtener listados de películas en cartelera, populares, mejor calificadas y próximas a estrenarse. Utiliza una API simple para el consumo de datos, Hooks y componentes adaptados para mostrar carruseles y listados. Además, la aplicación muestra información detallada de cada película, incluyendo la sinopsis y el elenco. Todo el desarrollo se realiza con React Native.

Gestionate

Esta aplicación es una herramienta sencilla diseñada para ayudar a trabajadores independientes a gestionar sus tareas diarias. Permite seleccionar un día en el calendario, registrar las actividades realizadas, indicar la duración en horas y establecer el monto a cobrar por cada tarea. Además, ofrece la posibilidad de generar un resumen detallado seleccionando un rango de fechas, lo que proporciona un desglose completo de las tareas realizadas y el total a cobrar. Todo el desarrollo ha sido realizado con React Native.
